Orbit View

Orbit view Orbit view is a web application that brings the cosmos closer to your finger tips. At the moment, Orbit View has the ability to view near-Earth asteroids, all of which having the possibility of being mined.

With an elegant user interface, Orbit View provides an easy-to-understand graphical representation of the asteroids of interest to the general public. To search for a particular feature like proximity from Earth, risk or a specific material, simply tick the relevant parameters or type in the search bar and Orbit View will display the information.

The interactive maps show the position of the object as well as its technical and commercial details. This information will shed some insight on future asteroid mining expeditions.

Currently, Orbit View contains the data for the top ten most viable asteroids for exploration. A careful assessment is conducted for each asteroid using information from public sources like Small Body Database Browser and Asterank.

The elements on the asteroids are currently predicted and estimated values. However, with the implementation of our probes and steady technical progression, the uncertainty will decrease with time.

Technical Details

Orbit View uses JsOrrery (MIT) open-source project as its core for rendering and displaying asteroids. The database and search bar are prototyped with bootstrap (MIT) with the cyborg theme (MIT). JQuery (MIT) is used throughout the application.

Future plans In the future, we hope that we will be able to create a comprehensive database of lucrative near-earth objects. By using Orbit View and a backend database, this application may eventually become the go-to place for key information on asteroids.

Check it out on http://astromine.space/ on the Orbit View tab!
